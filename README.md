# GFG DSA Problems

A curated collection of **GeeksforGeeks (GFG)** DSA problem solutions organized by difficulty and topic.

##  Repository Structure

GFG-DSA-Problems/
├── Basic/                    # Fundamentals and easy-level problems        
├── Easy/                     # Easy-level problem solutions
├── Medium/                   # Medium-level problem solutions
├── Hard/                     # Hard-level problem solutions
└── README.md                 # You're looking at it right now 🙂

##  What’s Inside?

- Solutions to selected **GFG practice problems**, organized neatly by difficulty.
- Modular structure helps in locating specific topics quickly.
- Code is meant for self-learning, easy comprehension, and practice.

##  How to Use This Repository

1. **Clone** the repository to your local machine:

   git clone https://github.com/shreyakesharwani-7/GFG-DSA-Problems.git

2. Navigate to the folder of the problem you’re interested in (e.g., `Basic/ArraySubset/`).

3. **Open** the solution file in your preferred editor or IDE.

4. **Run** the code and test with sample inputs as shown on GFG.

## Who Is This For?

* Yourself—for tracking your progress and keeping all solutions in one place.
* Learners reviewing topics or problem patterns.
* Anyone exploring foundational DSA via hands-on examples.
